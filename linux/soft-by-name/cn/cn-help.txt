Usage: cn [options] [command] [prompt]

Continue CLI - AI-powered development assistant. Starts an interactive session
by default, use -p/--print for non-interactive output.

Arguments:
  prompt                          Optional prompt to send to the assistant

Options:
  -v, --version                   Display version number
  --config <path>                 Configuration file for the assistant (can be a
                                  file path or hub slug)
  --org <slug>                    Organization slug to use for this session
                                  (supported only in headless mode)
  --readonly                      Start in plan mode (read-only tools)
  --auto                          Start in auto mode (all tools allowed)
  --verbose                       Enable verbose logging
  --beta-status-tool              Enable beta status tool
  --rule <rule>                   Add a rule (can be a file path, hub slug, or
                                  string content). Can be specified multiple
                                  times. (default: [])
  --mcp <slug>                    Add an MCP server from the hub (slug in format
                                  'owner/package'). Can be specified multiple
                                  times. (default: [])
  --model <slug>                  Add a model from the hub (slug in format
                                  'owner/package'). Can be specified multiple
                                  times. (default: [])
  --prompt <prompt>               Add to the initial user message (can be a file
                                  path, hub slug, or string content). Can be
                                  specified multiple times. (default: [])
  --allow <tool>                  Allow specified tool (overrides default
                                  policies). Can be specified multiple times.
                                  (default: [])
  --ask <tool>                    Ask for permission before using specified tool
                                  (overrides default policies). Can be specified
                                  multiple times. (default: [])
  --exclude <tool>                Exclude specified tool from use (overrides
                                  default policies). Can be specified multiple
                                  times. (default: [])
  --agent <slug>                  Load agent file from the hub (slug in format
                                  'owner/package')
  -p, --print                     Print response and exit (useful for pipes)
  --format <format>               Output format for headless mode (json). Only
                                  works with -p/--print flag.
  --silent                        Strip <think></think> tags and excess
                                  whitespace from output. Only works with
                                  -p/--print flag.
  --resume                        Resume from last session
  --fork <sessionId>              Fork from an existing session ID
  --beta-subagent-tool            Enable beta Subagent tool for invoking
                                  subagents
  -h, --help                      display help for command

Commands:
  login                           Authenticate with Continue
  logout                          Log out from Continue
  ls [options]                    List recent chat sessions and select one to
                                  resume
  remote-test [options] [prompt]  Test remote TUI mode with a local server
  review [options]                Run AI-powered reviews on your changes
