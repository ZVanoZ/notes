Использование:
  flatpak remotes [ПАРАМЕТР…]  - Показать удалённые репозитории

Параметры справки:
  -h, --help             Показать параметры справки

Параметры приложения:
  -u, --user             Работать с пользовательскими установками
  --system               Работать с установками для всех пользователей (по умолчанию)
  --installation=ИМЯ     Работать с установками для всех пользователей (без умолчаний)
  -d, --show-details     Показать подробности про удалённый репозиторий
  --show-disabled        Показать отключённые удалённые репозитории
  --columns=ПОЛЕ,…       Какую информацию показывать
  -v, --verbose          Показать отладочную информацию, -vv для подробных деталей
  --ostree-verbose       Показать отладочную информацию OSTree

Доступные столбцы:
  name            Показать имя
  title           Показать заголовок
  url             Показать адрес
  collection      Показать ID коллекции
  subset          Показать подмножество
  filter          Показать файл фильтра
  priority        Показать приоритет
  options         Показать параметры
  comment         Показать комментарий
  description     Показать описание
  homepage        Показать домашнюю страницу
  icon            Показать значок
  all             Показать все столбцы
  help            Показать доступные столбцы

Добавьте :s[tart], :m[iddle], :e[nd], :f[ull] для изменения эллепсии
