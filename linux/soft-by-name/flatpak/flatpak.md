## CMD

* help
````shell
flatpak --help &>flatpak-help.txt
````

* man
````shell
man flatpak &>flatpak-man.txt
````

* version
````shell
flatpak --version &>flatpak-version.txt
````

* flatpak install ... DroidCam
````shell
flatpak install flathub com.obsproject.Studio.Plugin.DroidCam
````
