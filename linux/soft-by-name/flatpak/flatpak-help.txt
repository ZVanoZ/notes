Использование:
  flatpak [ПАРАМЕТР…] КОМАНДА

Встроенные команды:
 Управление установленными приложениями и средами исполнения
  install                Установить приложение или среду исполнения
  update                 Обновить установленное приложение или среду исполнения
  uninstall              Удалить установленное приложение или среду исполнения
  mask                   Маскировать обновления и автоматическую установку
  pin                    Закрепить среду исполнения для предотвращения автоудаления
  list                   Показать список установленных приложений и/или сред исполнения
  info                   Показать информацию об установленных приложениях или средах исполнения
  history                Показать историю
  config                 Настроить flatpak
  repair                 Исправить установку flatpak
  create-usb             Поместить приложения или среды выполнения на съемный носитель

Поиск приложений или сред выполнения
  search                 Поиск удаленных репозиториев/сред выполнения

 Управление запуском приложений
  run                    Запустить приложение
  override               Переопределить разрешения для приложения
  make-current           Указать версию по умолчанию для запуска
  enter                  Войти в пространство имён запущенного приложения
  ps                     Перечислить запущенные приложения
  kill                   Приостановить запущенное приложение

 Управление доступом к файлам
  documents              Список экспортированных файлов
  document-export        Разрешить доступ приложения к указанному файлу
  document-unexport      Отозвать разрешение на доступ к указанному файлу
  document-info          Показать информацию об указанном файле

Изменить динамические разрешения
  permissions            Список разрешений
  permission-remove      Отозвать у приложения право на запись
  permission-set         Установить разрешения
  permission-show        Показать разрешения приложения
  permission-reset       Восстановить разрешения приложения

 Управление удалёнными репозиториями
  remotes                Список всех настроенных удалённых репозиториев
  remote-add             Добавить новый удалённый репозиторий (по АДРЕСУ)
  remote-modify          Изменить свойства настроенного удалённого репозитория
  remote-delete          Удалить настроенный удалённый репозиторий
  remote-ls              Список содержимого удалённого репозитория
  remote-info            Показать информацию об установленных приложениях или средах исполнения

 Сборка приложений
  build-init             Инициализировать каталог для сборки
  build                  Запустить команду сборки в каталоге сборки
  build-finish           Финализировать каталог сборки для экспорта
  build-export           Экспортировать из каталога сборки в репозиторий
  build-bundle           Создать файл с пакетом из каталога сборки локального репозитория
  build-import-bundle    Импортировать файл с пакетом
  build-sign             Подписать приложение или среду исполнения
  build-update-repo      Обновить файл со сводной информацией в репозитории
  build-commit-from      Создать новый коммит на базе существующей ссылки
  repo                   Получить информацию о репозитории

Параметры справки:
  -h, --help              Показать параметры справки

Параметры приложения:
  --version               Вывести информацию о версии и выйти
  --default-arch          Вывести архитектуру по умолчанию и выйти
  --supported-arches      Вывести поддерживаемые архитектуры и выйти
  --gl-drivers            Показать версию графических драйверов и выйти
  --installations         Вывести информацию о версии и выйти
  --print-updated-env     Вывести обновлённые переменные окружения, необходимые для запуска приложений flatpak
  --print-system-only     Включить в --print-updated-env только системные установки
  -v, --verbose           Показать отладочную информацию, -vv для подробных деталей
  --ostree-verbose        Показать отладочную информацию OSTree
