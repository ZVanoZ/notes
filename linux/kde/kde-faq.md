# [kde-faq.md](kde-faq.md)

Вопросы и ответы по KDE.


## Список

---

* Вопрос

```text
 Можно ли в KDE Plasma 5.27.5 для каждого монитора сделать свою панель задач?
Нужно чтобы на этой панели отображались те приложения, окна которых находятся на их мониторе.
При перемещении приложения на другой монитор, нужно перемещать вкладку на панель задач этого монитора. 
```

* Ответ

```text
Да, в KDE Plasma 5.27.5 можно настроить отдельные панели задач для каждого монитора, 
которые будут отображать только окна, находящиеся на соответствующем мониторе, и 
автоматически перемещать записи панели задач при перемещении окна.

KDE Plasma обладает высокой степенью настраиваемости, и эта функциональность реализуется с 
помощью виджета "Менеджер задач" (Task Manager) и его настроек.

Шаги по настройке отдельных панелей задач для каждого монитора

1.Добавьте панель на каждый монитор (если её нет):
* Щелкните правой кнопкой мыши по пустому месту на рабочем столе нужного монитора.
* Выберите "Добавить панель" (Add Panel).
* Выберите "Пустая панель" (Empty Panel) или "Панель по умолчанию" (Default Panel). 
Если вы хотите, чтобы панель на втором мониторе была точной копией основной, 
выберите "Панель по умолчанию" (хотя вам всё равно придётся настраивать виджет "Менеджер задач"). 
Если вы предпочитаете начать с чистого листа, выберите "Пустая панель".
* После добавления панели, возможно, вам потребуется настроить её 
положение (верх, низ, слева, справа) и размер. 
Для этого, наведите курсор на один из концов панели и нажмите на значок "Гамбургер" (три полоски), 
чтобы открыть "Параметры панели" (Panel Settings).

2. Добавьте виджет "Менеджер задач" на каждую панель:
* После того как панель добавлена, щелкните правой кнопкой мыши по пустому месту на этой новой панели.
* Выберите "Добавить виджеты" (Add Widgets).
* Найдите в списке виджет "Менеджер задач" (Task Manager). Перетащите его на панель. 
Если вы хотите, чтобы отображались только иконки, можете 
использовать "Менеджер задач: только значки" (Task Manager: Icons-Only).

3. Настройте "Менеджер задач" для отображения окон только с текущего монитора:
* Щелкните правой кнопкой мыши по пустому месту в области виджета "Менеджер задач" на 
панели (не по самой панели).
* Выберите "Настроить менеджер задач" (Configure Task Manager) или 
"Параметры менеджера задач" (Task Manager Settings).
* В открывшемся окне настроек найдите раздел "Поведение" (Behavior) или "Фильтры" (Filters).
* Установите флажок "Показывать только задачи с текущего экрана" (Show only tasks from the current screen).
* Нажмите "Применить" (Apply) или "ОК".

Что это даст:
* На каждой панели будет свой список окон: 
Виджет "Менеджер задач" на каждом мониторе будет отображать только те приложения, 
чьи окна находятся на этом конкретном мониторе.
* Автоматическое перемещение записи на панели: 
Когда вы перемещаете окно приложения с одного монитора на другой, запись этого окна автоматически 
исчезнет с панели задач первого монитора и появится на панели задач второго монитора. 
Это происходит без дополнительной настройки, так как менеджер задач фильтрует 
окна по их текущему экрану.

Важные замечания:
* Количество панелей: 
Вы можете иметь столько панелей, сколько хотите, и размещать их в любом месте на любом мониторе.
* Персонализация: 
Каждая панель настраивается независимо. 
Это означает, что вы можете иметь разные виджеты, размеры и темы для панелей на разных мониторах.
* Контекстное меню: 
Важно различать контекстное меню самой панели (правый клик по пустому месту на панели) и 
контекстное меню виджета "Менеджер задач" (правый клик по пустому месту внутри области 
виджета "Менеджер задач"). 
Настройки, которые вам нужны, находятся в настройках виджета.

Эта функциональность является одной из сильных сторон KDE Plasma в работе с несколькими 
мониторами, предлагая гибкость и удобство для пользователей.
```
