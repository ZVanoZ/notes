# [checkbox.md](checkbox.md)

Изучение чекбоксов

---

Маркированный список через "*"

* [ ] Земля
* [x] Луна
* [ ] Марс

---

Без списка нельзя

[ ] Молоко
[x] Хлеб
[ ] Яйца

---

Маркированный список через "-"

- [x] Задача 1
- [ ] Задача 2
- [ ] Задача 3

---

Нумерованный список не выводит цифры 

1. [ ] Париж
2. [x] Рим
3. [ ] Барселона
---

Правильный способ нумерованных списков с чекбоксами

1. * [ ] Париж
2. * [x] Рим
3. * [ ] Барселона
