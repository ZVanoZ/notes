## demo 2 - markdown внутри блока

<details>
  <summary>детально...</summary>

Это содержимое сворачиваемого блока.

Вы можете поместить сюда любой **Markdown-контент**:

* Пункт 1
* Пункт 2

Пример кода:

```js
echo("Привет, мир!")
```

</details>
