# react-window

"react-window" - библиотека компонент для отображения больших списков.

Решает проблему производительности при отображении больших массивов данных.

Лицензия: "MIT"

Группы компонент:
* List
* Table
* Grid



## Ссылки

* https://github.com/bvaughn/react-window

---

## Установка 

```shell
npm install react-window
```
