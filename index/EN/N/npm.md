## RAW

````bash
# Вызываем консольный диалог для создания "package.json" в текущей директории.
npm init

# Получить помощь по команде install
npm help install

# Установить пакет "node-gyp" в глобальную область 
npm install -g node-gyp
# Установить пакет "node-gyp" в локальную область
## См. "./node_modules/node-gyp/bin/node-gyp.js" 
npm install node-gyp
# Удалить пакет "node-gyp" из локальной области
npm remove node-gyp
````
