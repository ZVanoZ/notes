Расскажи про Comntext7 MCP

---

**Context7 MCP** — это один из самых полезных инструментов для разработчиков, использующих ИИ (особенно в Cursor, VS Code или Claude Desktop). Это специализированный сервер, работающий по протоколу **Model Context Protocol (MCP)**, который решает проблему «устаревших знаний» нейросетей.

Вот всё, что нужно о нем знать:

### 1. Что это такое?

Это «мост» между вашим ИИ-редактором и **актуальной документацией** популярных библиотек и фреймворков.
Обычно база знаний моделей (вроде Claude 3.5 или GPT-4) ограничена датой их обучения (например, серединой 2024 года). Если за это время библиотека (например, `Next.js` или `Tailwind`) обновилась, ИИ может выдавать несуществующие методы. **Context7** подтягивает свежую документацию прямо в момент вашего запроса.

### 2. Ключевые возможности

* **Актуальность:** Он сканирует официальные источники и возвращает документацию для конкретных версий библиотек.
* **Примеры кода:** Вместе с описанием API он выдает свежие примеры реализации, соответствующие текущим стандартам.
* **Умный поиск:** Вы можете просто спросить: «Как реализовать X в библиотеке Y?», и Context7 найдет нужный раздел документации за вас.

### 3. Как это работает в Cursor

Когда вы устанавливаете Context7 как MCP-сервер, у Агента появляются новые «инструменты»:

1. **`resolve-library-id`**: ИИ находит ID нужной библиотеки (например, `/nextjs/docs`).
2. **`query-docs`**: ИИ делает запрос к этой документации, получает текст и на его основе пишет вам код.

### 4. Как установить (для Cursor)

1. Зайдите в **Settings** (`Ctrl + Shift + J`) -> **General** -> **Features**.
2. Найдите раздел **MCP**.
3. Нажмите **+ Add New MCP Server**.
4. Введите параметры:
* **Name:** `context7`
* **Type:** `command`
* **Command:** `npx -y @upstash/context7-mcp`


5. Нажмите **Save**.

### 5. Как пользоваться

Просто скажите Агенту в чате:

> *"Используй context7, чтобы найти документацию по последней версии `shadcn/ui` и помоги мне создать форму авторизации."*
