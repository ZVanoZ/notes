# About Cursor

## 1. Общее описание

**Cursor** — это ИИ-ориентированный редактор кода (AI Code Editor), построенный на базе **VSCodium** (свободная версия VS Code). Программа разработана для максимизации продуктивности программиста за счет глубокой интеграции больших языковых моделей (LLM) непосредственно в процесс написания кода.

**Ключевые возможности:**

* **Tab-complete:** Интеллектуальное автодополнение целых блоков кода.
* **Chat:** Контекстный чат с ИИ, который видит весь проект (`@Codebase`).
* **Composer:** Инструмент для автоматической генерации или правки нескольких файлов одновременно.
* **MCP Support:** Поддержка протокола Model Context Protocol для расширения возможностей ИИ внешними инструментами.

## 2. Лицензия

Cursor распространяется под **проприетарной лицензией** (Proprietary).

* Вы можете использовать его бесплатно (с ограничениями по количеству запросов к ИИ).
* Для профессионального использования предусмотрены платные подписки (Pro и Business).
* Несмотря на то, что он базируется на компонентах с открытым кодом, сам «интеллектуальный слой» и серверная часть являются частной собственностью компании Anysphere.

## 3. Статус Open Source

Программа **не является Open Source**.
Хотя Cursor использует форк VS Code (который открыт), ключевые функции (AI-движок, индексация кода, проприетарные алгоритмы предсказания правок) закрыты.
